Bilisound 提供了全面的数据管理功能，让您可以轻松备份和恢复您的歌单数据，以及保存您喜爱的音频文件到本地设备。本指南将详细介绍这些功能的使用方法。

## 歌单备份与还原

Bilisound 支持将您精心整理的歌单导出为文件，以及从文件中导入歌单数据。这使您能够：

- 在设备之间迁移您的歌单
- 安全备份您的歌单数据
- 与朋友分享您的歌单

### 导出单个歌单

您可以轻松导出单个歌单：

1. 进入「歌单」页面
2. 长按您想要导出的歌单
3. 在弹出的菜单中选择「导出」
4. 选择保存位置，歌单将以 TOML 格式导出

导出的歌单文件包含该歌单的所有信息，包括歌单名称、描述以及所有音频项目的详细信息。

### 导出全部歌单

如果您想一次性备份所有歌单：

1. 进入「设置」页面
2. 选择「数据管理」选项
3. 点击「导出全部歌单」按钮
4. 选择保存位置，所有歌单将打包导出为一个 TOML 文件

### 导入歌单

要恢复之前导出的歌单：

1. 进入「歌单」页面
2. 点击右上角加号按钮，选择「导入歌单」
3. 选择之前导出的 TOML 格式歌单文件
4. 确认导入

> **注意**：导入歌单时，如果存在同名歌单，两个同名的歌单会同时保留。您可以视情况删除重复的歌单，或者将一个歌单中的部分曲目复制到另一个歌单中。

## 音频文件备份

除了歌单数据，Bilisound 还支持将离线缓存的音频文件保存到本地设备，方便您在其它设备上播放或者长期保存。

### 从视频详情页保存音频

在浏览视频详情页时：

1. 长按您想要保存的分集
2. 在弹出的菜单中选择「保存到文件」。如果只有「缓存」选项，则先点击它，等待音频缓存完成
3. 选择保存位置

### 从播放器界面保存音频

在播放音频时：

1. 点击播放器界面右上角的菜单按钮
2. 在弹出的菜单中选择「保存到文件」。如果只有「缓存」选项，则先点击它，等待音频缓存完成
3. 选择保存位置

## 数据安全与隐私

Bilisound 的所有数据备份功能都在本地设备上进行，不会将您的数据上传到任何服务器。导出的歌单文件和保存的音频文件仅存储在您指定的位置。

## 常见问题

### 我可以在不同平台之间迁移歌单吗？

是的，Bilisound 的歌单导出格式在所有支持的平台上都是通用的。您可以在 Android、iOS 和 Web 版本之间自由迁移您的歌单数据。

### 导出的歌单文件占用空间大吗？

歌单导出文件仅包含元数据信息，不包含实际的音频内容，因此文件通常很小，只有几十到几百 KB。TOML 格式的文件结构清晰，便于阅读和编辑。

### 保存的音频文件格式是什么？

Bilisound 保存的音频文件格式是 m4a 格式。
