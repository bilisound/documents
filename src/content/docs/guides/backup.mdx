Bilisound 让数据管理变得简单。无论是备份歌单，还是将喜欢的音频保存到本地，您都可以轻松完成。

## 备份与还原歌单

您可以将歌单导出为文件，或者从文件中导入数据。这方便您：

- 在不同设备间迁移歌单
- 备份歌单防止丢失
- 与朋友分享精彩歌单

### 导出单个歌单

导出单个歌单非常简单：

1. 前往「歌单」页面
2. 长按需要导出的歌单
3. 在菜单中选择「导出」
4. 选择保存位置，文件将以 TOML 格式保存

导出的文件包含歌单的名称、描述以及所有音频的详细信息。

### 导出全部歌单

如果您想一次性备份所有数据：

1. 前往「设置」
2. 进入「数据管理」
3. 点击「导出全部歌单」
4. 选择保存位置，所有歌单将打包为一个 TOML 文件导出

### 导入歌单

如何还原歌单：

1. 前往「歌单」页面
2. 点击右上角的「+」号，选择「导入歌单」
3. 选择之前的 TOML 备份文件
4. 确认导入即可

> **注意**：导入时如果存在同名歌单，系统会保留两者。您可以手动删除重复项，或整理合并曲目。

## 备份音频文件

除了歌单，您还可以将缓存的音频文件保存到本地。这样方便您在其他设备播放，或进行长期收藏。

### 从详情页保存

在视频详情页：

1. 长按目标分集
2. 选择「保存到文件」（若只有「缓存」选项，请先点击缓存，待完成后再操作）
3. 选择保存位置

### 从播放器保存

播放音频时：

1. 点击播放器右上角的菜单
2. 选择「保存到文件」（需先完成缓存）
3. 选择保存位置

## 数据安全与隐私

Bilisound 的数据操作均在本地进行，不会上传至任何服务器。导出的文件仅存储在您指定的位置，安全可控。
